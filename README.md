# Amazon Scraper Backend

Server-side Amazon product scraper backend deployed on Vercel. Extracts brand, packaging, and product details from Amazon URLs.

## Features

- Fetches Amazon product pages server-side
- - Extracts brand name, packaging info, and product images
  - - Returns structured JSON data
    - - CORS-friendly API endpoint
